# webtk

Web based UI toolkit match native system theme

```jsx
import { App, StatusBar, ToolBar, Window } from 'webtk';

<App theme="breeze-dark">
  <Window title="My App" style={{ maxWidth: 800 }}>
    <ToolBar>TODO</ToolBar>
    <div>TODO</div>
    <StatusBar>TODO</StatusBar>
  </Window>
</App>;
```
